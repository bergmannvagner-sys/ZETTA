# Cores Oficiais

## Tokens

- Deep Midnight: `#0A0F1F`
- Core Violet: `#8A28E2`
- Soft Lilac: `#B89BFF`
- Calm Cyan: `#00E5FF`
- Presence Glow: `#0D47FF`
- Magenta Accent: `#FF4DFF`

## Uso

Deep Midnight deve ser a base dominante.

Calm Cyan deve sinalizar ação primária, escuta e clareza.

Core Violet e Soft Lilac devem sinalizar presença, acolhimento, privacidade e cuidado emocional.

Magenta deve ser usado com moderação, apenas como brilho complementar.

## Evitar

- Vermelho agressivo
- Verde neon
- Amarelo forte
- Paletas gamer
- Branco excessivo
- Cards claros na Home

## Segurança emocional

Estados de crise devem usar menos saturação, menos glow e menos movimento. O visual deve acalmar, não alarmar.
